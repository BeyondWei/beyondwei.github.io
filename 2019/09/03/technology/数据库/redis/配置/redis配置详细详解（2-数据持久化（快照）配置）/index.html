redis配置详细详解（2.数据持久化（快照）配置）

觉可以忘睡，饭可以忘吃，博客不能忘更

save

  • 模板：save 900 1
  • 作用：redis数据库持久化条件（900秒钟有一次操作就会持久化一次）

stop-writes-on-bgsave-error

  • 模板：stop-writes-on-bgsave-error yes
  • 作用：当redis持久化失败的时候不允许在写入

rdbcompression

  • 模板：rdbcompression yes
  • 作用：在转储.rdb数据持久化时使用LZF压缩字符串对象

rdbchecksum

  • 模板：rdbchecksum yes
  • 作用：是否开启在持久化时对字符串的校验

dbfilename

  • 模板：dbfilename dump.rdb
  • 作用：持久化文件的名称

dir

  • 模板：dir /var/lib/redis
  • 作用：指定持久化文件的地址
