创建博客

博客安装

安装准备

软件/工具准备

  1. hexo
  2. MacOs或者Windows
  3. nodejs
  4. github(git)

安装步骤

安装nodejs

前往官网下载nodejs，地址：
http://nodejs.cn/download/

安装cnpm

nodejs安装完毕自带npm，但是国外镜像源速度慢，建议下载cnpm并配置淘宝的镜像源，下载并配置命令：
npm install -g cnpm --registry=https://registry.npm.taobao.org

配置Git

MacOs 用户自带Git工具无需配置
Windows 用户下载Git后，配置Git的环境变量（不会自行百度）

下载hexo

使用cnpm下载hexo，下载指令：
cnpm install -g hexo-cli

安装hexo

创建一个空的目录(bolg，文件名自己定义)，Windows用户建议在C盘目录下，以免产生奇奇怪怪的问题，
打开终端(Windows打开cmd),前往创建的目录，初始化一个hexo的实例，初始化指令：
sudo hexo init
（注意在bolg文件夹内，如果出问题大不了把文件删了重新来）

访问本地hexo

通过命令hexo s启动hexo，hexo默认端口为4000 ；
访问localhost:4000 访问

发布至GitHub

能够正常访问就是安装成功了，加下去把hexo部署至GitHub

创建项目

没有gitHub账号的自行创建；
创建新的项目
项目名称必须以 github.io 结尾；
例如：
zhangsan.github.io

将hexo推送到GitHub

下载部署工具，安装命令：
cnpm install --save hexo-deployer-git
进入bolg目录找到_config.yml 配置文件；
更改文件配置内容
deploy:
type: git
repo: https://github.com/XXXX/xxx.github.io.git
branch: master
保存后退出
使用hexo d命令发布至GitHub
通过访问 xxx.github.io.git 查看是否部署成功

创建新的博客

通过命令创建博客:
hexo new "My New Post"
创建完bolg后自行在文件中进行编辑博客内容，使用mrakdown语法编写；
通过hexo clean 删除缓存在通过hexo d 重新发布
